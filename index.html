⚠️ Reality check on "80% accuracy"
Real swing strategies run 45–60% win rates. Anything above 70% on broad markets is almost always overfit or survivorship bias. This tool ranks by Sharpe ratio (risk-adjusted return), not win rate. A 55% strategy at 2:1 reward:risk beats an 80% strategy at 1:3. If a result looks too good, it'll get flagged below.
Data: Yahoo Finance via free CORS proxy (broker APIs can't be called from a browser). Must be hosted on an https link. Daily OHLCV matches what Upstox gives for swing trading.
📋 Offline mode — paste your own CSV (guaranteed to work)
If proxies are down, export daily data from any source and paste it here. Format per line: SYMBOL,YYYY-MM-DD,open,high,low,close,volume. One symbol or many; needs ≥260 rows per symbol. Then tap Run on pasted data.
